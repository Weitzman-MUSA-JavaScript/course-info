Let's discuss custom semantics

Custom semantics are a way to add meaning to your HTML by creating your own element classes. This can be useful for creating reusable components or for adding meaning to your content that isn't already covered by the standard HTML elements.

For example, you could create a legend class and apply common legend styling. This would allow you to easily reuse this class throughout your site without having to duplicate the css attributes each time.
